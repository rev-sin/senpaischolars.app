# Senpai Scholars

## Getting Started

Assuming you have pnpm installed.

```bash
pnpm i
pnpm dev

# add packages
pnpm add <package-name>
```

Checkout package.json for scripts and other miscellaneous configs.

## Locally

> All biome checks must pass locally!\
> The app must build!\
> The app will be formatted pre-commit!

## North
